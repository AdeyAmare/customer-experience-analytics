# Database Directory

This folder contains database-related files for the project, including the schema and query results used for analysis or verification.

## Contents

### 1. `schema.sql`
- **Purpose:** Defines the database structure for the project.  
- **Contents:** SQL commands to create tables such as `banks` and `reviews`, set up constraints, indexes, and relationships.  
- **Effect:** Provides a blueprint to initialize or reset the PostgreSQL database.

### 2. `query_results/`
- **Purpose:** Stores results from important or frequently run queries.  
- **Contents:** CSV or SQL export files of query results for reporting, validation, or testing purposes.  
- **Effect:** Allows easy access to key insights without re-running queries on the live database.
